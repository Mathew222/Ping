# Notification Sounds

This directory contains notification sound files for the Ping app.

## Sound Files

1. **gentle_chime.mp3** - Soft, pleasant chime (default)
2. **classic_bell.mp3** - Traditional notification bell
3. **digital_beep.mp3** - Modern digital sound
4. **urgent_alert.mp3** - For important reminders

## Adding Custom Sounds

To add your own notification sounds:

1. Find or create MP3 files (1-3 seconds duration, < 500KB)
2. Name them according to the list above
3. Place them in this directory
4. Rebuild the app

## Free Sound Resources

You can find free notification sounds at:
- https://pixabay.com/sound-effects/search/notification/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

## Note

For now, the app will use Android/iOS system default sounds. Replace these placeholder files with actual MP3 files for custom sounds.
